# 项目简介

## 核心功能
- 基于YOLOv8的昆虫检测系统，支持对10类昆虫进行识别
- 提供完整源码、标注数据集、训练好的模型及训练结果
- 支持图片、视频及摄像头进行实时检测
- 界面显示目标位置、目标总数、置信度等信息
- 支持批量检测功能，可直接查看所有检测结果
- 支持检测结果保存
- 可自定义置信度和IOU阈值
- 支持选择显示标签和原图
- 可选择不同的检测模型

## 适用场景
- 农业昆虫监测与识别
- 农作物病虫害防治
- 生态环境研究
- 昆虫分类学研究
- 农业自动化管理

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda作为开发环境
- 依赖Python 3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 包含完整的UI界面源文件和图标
- 内置1000张昆虫数据集，包括军虫、豆科水泡甲虫、红蜘蛛等10类昆虫
- 配置成功环境后可直接使用
